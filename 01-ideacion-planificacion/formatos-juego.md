# Formatos de Juego para Actividades

Catálogo de formatos basados en concursos de TV, adaptados para refuerzo de lenguas (2º ESO).

---

## Formatos disponibles

### 1. La Ruleta

**Inspirado en:** La Ruleta de la Suerte

**Mecánica:**
- Panel con palabra/frase oculta (solo se ven huecos)
- Se da una pista (definición, contexto, sinónimo)
- Los equipos piden letras o intentan resolver
- La ruleta añade azar: pierde turno, bonus x2, quiebra

**Ideal para:** Vocabulario, expresiones, refranes

---

### 2. El Cazador Lingüístico

**Inspirado en:** El Cazador

**Mecánica:**
- Preguntas rápidas tipo test (3-4 opciones)
- Tablero de avance: el alumno/equipo vs. el "cazador"
- Si aciertas, avanzas; si fallas, el cazador se acerca
- Si el cazador te alcanza, eliminado

**Ideal para:** Ortografía, gramática, comprensión lectora

---

### 3. Millonario por Equipos

**Inspirado en:** ¿Quién quiere ser millonario?

**Mecánica:**
- Preguntas con 4 opciones, dificultad creciente
- Comodines: 50%, consulta al equipo, pista del profe
- Por equipos: 30 segundos para deliberar
- Escalada de "premios" (puntos o niveles)

**Ideal para:** Cualquier contenido (muy versátil)

---

### 4. Password

**Inspirado en:** Password

**Mecánica:**
- Un alumno ve la palabra secreta
- Debe dar pistas (1 palabra por turno) sin decirla
- Su equipo intenta adivinar
- Límite de pistas o tiempo

**Ideal para:** Vocabulario, sinónimos, expresión oral

## Matriz de compatibilidad

| Formato | Ortografía | Vocabulario | Comprensión | Gramática | Expresión |
|---------|:----------:|:-----------:|:-----------:|:---------:|:---------:|
| **Ruleta** | Regular | Excelente | - | - | - |
| **Cazador** | Excelente | Bueno | Bueno | Excelente | - |
| **Millonario** | Excelente | Bueno | Bueno | Excelente | Regular |
| **Password** | - | Excelente | - | - | Bueno |

### Leyenda
- **Excelente:** El formato potencia este contenido
- **Bueno:** Funciona bien
- **Regular:** Posible pero no óptimo
- **-:** No recomendado

## Guía rápida: Qué formato elegir

| Si quieres trabajar... | Formato principal | Alternativa |
|------------------------|-------------------|-------------|
| Ortografía (b/v, g/j, h, tildes) | Cazador | Millonario |
| Vocabulario (definiciones, campos) | Ruleta | Password |
| Comprensión lectora | Cazador | Millonario |
| Gramática básica | Cazador | Millonario |
| Expresión oral | Password | - |
| Refranes y expresiones | Ruleta | - |
| Sinónimos/antónimos | Password | Ruleta |
