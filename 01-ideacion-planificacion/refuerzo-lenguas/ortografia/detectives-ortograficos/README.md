# Detectives Ortográficos

## Descripción

Los alumnos son **detectives ortográficos** que investigan errores en "tweets" y publicaciones de redes sociales. Enfoque inductivo: descubren el error, formulan la regla, aplican la corrección.

## Objetivos

- Identificar errores ortográficos frecuentes
- Deducir reglas ortográficas a partir de ejemplos
- Desarrollar conciencia ortográfica en contextos reales

## Errores trabajados

| Error | Ejemplo incorrecto | Corrección |
|-------|-------------------|------------|
| a ver / haber | "Haber si vienes" | "A ver si vienes" |
| hay / ahí / ay | "Ay mucha gente" | "Hay mucha gente" |
| vaya / valla | "Que te valla bien" | "Que te vaya bien" |
| echo / hecho | "Ya lo he echo" | "Ya lo he hecho" |
| k / c | "Ke ases" | "Qué haces" |
| Tildes interrogativas | "Cuando vienes?" | "¿Cuándo vienes?" |
| ha / a | "Voy ha casa" | "Voy a casa" |
| echar (sin h) | "Te hecho de menos" | "Te echo de menos" |
| por qué / porque / no sé | "Nose porque" | "No sé por qué" |
| iba (sin h) | "Hiba al cine" | "Iba al cine" |

## Materiales necesarios

- Proyector
- Presentación con tweets (PowerPoint o PDF)
- Papel para anotar (opcional)

## Duración

45 minutos (1 sesión)
