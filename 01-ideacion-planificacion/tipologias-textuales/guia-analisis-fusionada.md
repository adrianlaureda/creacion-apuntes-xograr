# Guía de análisis: Las 4 tipologías textuales

## Instrucciones

Tienes 4 textos: A, B, C y D. Analízalos completando las dos tablas siguientes.

---

## Tabla 1: Textos A y B (Narrativo vs Descriptivo)

| Aspecto | Texto A | Texto B |
|---------|---------|---------|
| **1.** ¿Qué hace este texto: cuenta algo o muestra cómo es algo? | | |
| **2.** ¿Aparecen personajes? ¿Sucede algo (hay historia)? | | |
| **3.** ¿Qué verbos predominan: de acción (correr, decir) o de estado (ser, estar)? | | |
| **4.** ¿Hay muchos adjetivos? Pon un ejemplo. | | |

**Conclusión:** ¿Cuál es narrativo y cuál descriptivo?

- Narrativo: Texto _____ porque _________________________________

- Descriptivo: Texto _____ porque _________________________________

---

## Tabla 2: Textos C y D (Expositivo vs Argumentativo)

| Aspecto | Texto C | Texto D |
|---------|---------|---------|
| **1.** ¿Qué hace este texto: explica algo o defiende una idea? | | |
| **2.** ¿Hay una opinión clara del autor? ¿Cuál? | | |
| **3.** ¿Intenta convencerte de algo? ¿De qué? | | |
| **4.** ¿Usa datos objetivos o razones personales para apoyarse? | | |

**Conclusión:** ¿Cuál es expositivo y cuál argumentativo?

- Expositivo: Texto _____ porque _________________________________

- Argumentativo: Texto _____ porque _________________________________
