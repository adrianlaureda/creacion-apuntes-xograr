# Tipologías Textuales - Unidad Didáctica

## Descripción

Unidad didáctica para 4º ESO sobre las cuatro tipologías textuales básicas: narrativo, descriptivo, expositivo y argumentativo.

## Enfoque metodológico

**Inductivo y comparativo:**
- Los alumnos ya tienen contacto previo con estas tipologías
- Se trabajan en pares contrastados, no de forma aislada
- Bloque 1: Narrativo vs Descriptivo (lo que "cuenta" vs lo que "muestra")
- Bloque 2: Expositivo vs Argumentativo (lo que "explica" vs lo que "defiende")
- Síntesis final con las 4 tipologías

## Duración

3 semanas (9 sesiones de ~40 minutos efectivos)
