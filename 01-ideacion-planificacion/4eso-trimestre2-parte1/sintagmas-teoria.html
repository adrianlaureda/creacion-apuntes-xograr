¿Qué es un sintagma?

Un sintagma es un grupo de palabras que se organizan alrededor de una palabra principal llamada núcleo.

Las demás palabras del sintagma son modificadores del núcleo.

Regla de oro

El tipo de sintagma depende de la categoría gramatical del núcleo.

Los 4 tipos básicos

SN
Sintagma Nominal

Núcleo: Sustantivo

SAdj
Sintagma Adjetival

Núcleo: Adjetivo

SAdv
Sintagma Adverbial

Núcleo: Adverbio

SPrep
Sintagma Preposicional

Núcleo: Preposición + SN

Estructura básica

Cualquier sintagma sigue esta fórmula general:

SINTAGMA = (MOD) + NÚCLEO + (MOD)
Abreviatura Significado Ejemplo
N Núcleo (palabra principal) casa, alto, lejos
MOD Modificador (acompaña al núcleo) grande, muy, de madera
( ) Elemento opcional —

Sintagma Nominal (SN)

El SN tiene como núcleo un sustantivo (o pronombre). Es el sintagma más frecuente.

SN = (Det) + N + (MOD)
Componente Qué es Ejemplos
Det Determinante el, la, un, mi, este, tres
N Sustantivo casa, perro, María
MOD Modificador grande, de madera

SN: Ejemplos graduados

  • Básico el perro
    → Det + N
  • Intermedio el perro negro
    → Det + N + MOD (adj)
  • Avanzado (nivel examen) el coche rojo de mi padre
    → Det + N + MOD (adj) + MOD (SPrep)

El MOD puede ser

Un adjetivo (negro, rojo) o un SPrep (de mi padre, de madera).

Sintagma Adjetival (SAdj)

El SAdj tiene como núcleo un adjetivo. Suele funcionar como modificador de un sustantivo o como atributo.

SAdj = (MOD) + N + (MOD)
Componente Qué es Ejemplos
N Adjetivo alto, inteligente, feliz
MOD Adverbio o SPrep muy, bastante, de matemáticas

SAdj: Ejemplos graduados

  • Básico alto
    → N
  • Intermedio muy alto
    → MOD (adv) + N
  • Avanzado (nivel examen) muy orgulloso de su trabajo
    → MOD (adv) + N + MOD (SPrep)

El MOD puede ser

Un adverbio (muy, bastante, demasiado) o un SPrep (de su trabajo).

Sintagma Adverbial (SAdv)

El SAdv tiene como núcleo un adverbio. Indica circunstancias (tiempo, lugar, modo, cantidad...).

SAdv = (MOD) + N + (MOD)
Componente Qué es Ejemplos
N Adverbio lejos, ayer, bien, mucho
MOD Otro adverbio o SPrep muy, de aquí

SAdv: Ejemplos graduados

  • Básico lejos
    → N
  • Intermedio muy lejos
    → MOD (adv) + N
  • Avanzado (nivel examen) muy lejos de aquí
    → MOD (adv) + N + MOD (SPrep)

El MOD puede ser

Otro adverbio (muy, bastante) o un SPrep (de aquí, de mi casa).

Sintagma Preposicional (SPrep)

El SPrep comienza siempre por una preposición seguida de un SN.

SPrep = E (preposición) + T (SN)
Componente Qué es Ejemplos
E Enlace (preposición) a, de, en, con, para, por, sin
T Término (SN) la mesa, mi casa, él

SPrep: Ejemplos graduados

  • Básico en casa
    → E + T (N)
  • Intermedio para mi hermano
    → E + T (Det + N)
  • Avanzado (nivel examen) desde el parque grande
    → E + T (Det + N + MOD)

Importante

El término siempre es un SN, así que se analiza como tal (Det + N + MOD...).

Cómo identificar cada sintagma

Si el núcleo es... El sintagma es... Ejemplo rápido
Un sustantivo SN ● el libro, María
Un adjetivo SAdj ■ muy alto, feliz
Un adverbio SAdv ▲ bastante lejos, ayer
Una preposición + SN SPrep ◆ de madera, en casa

Abreviaturas

Abrev. Significado
Det Determinante
N Núcleo
MOD Modificador
E Enlace (preposición)
T Término

Determinantes comunes

Artículos: el, la, los, las, un, una

Posesivos: mi, tu, su, nuestro

Demostrativos: este, ese, aquel

Numerales: un, dos, tres, primero

Indefinidos: algún, ningún, cada

Consejos para el examen

1
Localizar sintagmas

Identifica el núcleo primero. Después busca los límites: ¿dónde empieza y termina?

2
Analizar estructura

Subraya el núcleo. Identifica Det (si hay sustantivo). Busca los MOD y di de qué tipo son.

3
No olvides el SPrep

Si el MOD es un SPrep, analiza también su estructura interna: E + T.

4
Error frecuente

No confundas el tipo de sintagma. Fíjate en el núcleo, no en los modificadores.
