>

>

>
> Podemos ser <u>moderadamente optimistas</u>. **Esa** esperanza se basa en datos reales: la sociedad está despertando <u>ante la crisis climática</u>.

## Ejercicio 3: Sintagmas - Texto 1

**a) una realidad innegable** → SN
- Det (*una*) + N (*realidad*) + MOD (*innegable*, adjetivo)

**b) el calentamiento global** → SN
- Det (*el*) + N (*calentamiento*) + MOD (*global*, adjetivo)

**c) medidas urgentes** → SN
- N (*medidas*) + MOD (*urgentes*, adjetivo)

**d) bolsas reutilizables** → SN
- N (*bolsas*) + MOD (*reutilizables*, adjetivo)

**e) en el último año** → SPrep
- E (*en*) + T [SN: Det (*el*) + MOD (*último*) + N (*año*)]

**f) moderadamente optimistas** → SAdj
- MOD (*moderadamente*, adverbio) + N (*optimistas*)

**g) ante la crisis climática** → SPrep
- E (*ante*) + T [SN: Det (*la*) + N (*crisis*) + MOD (*climática*)]

## Notas para el profesor

- Los textos permiten trabajar ambos contenidos de forma integrada
- Se pueden hacer por partes: primero cohesión de ambos textos, luego sintagmas
- O alternar: Texto 1 completo, después Texto 2 completo

- Aceptar variantes en nomenclatura de conectores si son coherentes
