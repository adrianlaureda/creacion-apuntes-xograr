# Sintagmas: Guía de 4º ESO

## ¿Qué es un sintagma?

Un **sintagma** es un grupo de palabras que se organizan alrededor de una palabra principal llamada **núcleo**. Las demás palabras del sintagma son **modificadores** del núcleo.

**Regla de oro:** El tipo de sintagma depende de la categoría gramatical del núcleo.

---

## Los 4 tipos básicos

| Tipo | Núcleo | Símbolo | Color |
|------|--------|---------|-------|
| **SN** (Sintagma Nominal) | Sustantivo | ● círculo | Azul |
| **SAdj** (Sintagma Adjetival) | Adjetivo | ■ cuadrado | Verde |
| **SAdv** (Sintagma Adverbial) | Adverbio | ▲ triángulo | Naranja |
| **SPrep** (Sintagma Preposicional) | Preposición + SN | ◆ rombo | Violeta |

---

## Estructura básica de cualquier sintagma

```
SINTAGMA = (MOD) + NÚCLEO + (MOD)
```

- **N** = Núcleo (palabra principal)
- **MOD** = Modificador (cualquier elemento que acompaña al núcleo)
- Los paréntesis indican elementos opcionales

El modificador puede ser de distintos tipos según su categoría:
- Un adjetivo solo
- Un sintagma adverbial
- Un sintagma preposicional
- Etc.

La clave está en identificar **qué tipo de sintagma** es ese modificador.

---

## 1. Sintagma Nominal (SN) ●

### Definición

El sintagma nominal tiene como núcleo un **sustantivo** (o un pronombre). Es el sintagma más frecuente.

### Estructura

| Componente | Abreviatura | Qué es | Ejemplos |
|------------|-------------|--------|----------|
| Determinante | **Det** | Presenta al sustantivo | el, la, un, mi, este, tres |
| Núcleo | **N** | Sustantivo (palabra principal) | casa, perro, María, libertad |
| Modificador | **MOD** | Cualquier elemento que complementa al N | grande, de madera, muy interesante |

### Fórmula

```
SN = (Det) + N + (MOD)
```

### Tipos de modificadores en el SN

El modificador puede ser:
- **Un adjetivo:** *el perro **negro*** → MOD = adjetivo
- **Un SPrep:** *el coche **de mi padre*** → MOD = SPrep (E + T)
- **Otro SN:** *Madrid, **capital de España*** → MOD = SN (aposición)

### Ejemplos graduados

**Básico:**
- *el perro* → Det + N
- *María* → N (nombre propio, sin determinante)

**Intermedio:**
- *el perro negro* → Det + N + MOD (adjetivo)
- *mi hermana pequeña* → Det + N + MOD (adjetivo)

**Avanzado (nivel examen):**
- *el coche rojo de mi padre* → Det + N + MOD (adj) + MOD (SPrep)
- *la casa antigua del pueblo* → Det + N + MOD (adj) + MOD (SPrep)

**Análisis del MOD cuando es SPrep:**
- *de mi padre* → SPrep: E (*de*) + T (*mi padre*, que es un SN: Det + N)

---

## 2. Sintagma Adjetival (SAdj) ■

### Definición

El sintagma adjetival tiene como núcleo un **adjetivo**. Suele funcionar como modificador de un sustantivo o como atributo.

### Estructura

| Componente | Abreviatura | Qué es | Ejemplos |
|------------|-------------|--------|----------|
| Núcleo | **N** | Adjetivo (palabra principal) | alto, inteligente, feliz |
| Modificador | **MOD** | Cualquier elemento que acompaña al adjetivo | muy, bastante, de matemáticas |

### Fórmula

```
SAdj = (MOD) + N + (MOD)
```

### Tipos de modificadores en el SAdj

El modificador puede ser:
- **Un adverbio:** *muy contento* → MOD = SAdv (o adverbio solo)
- **Un SPrep:** *orgulloso de su trabajo* → MOD = SPrep

### Ejemplos graduados

**Básico:**
- *alto* → N
- *muy alto* → MOD (adv) + N

**Intermedio:**
- *bastante inteligente* → MOD (adv) + N
- *demasiado difícil* → MOD (adv) + N

**Avanzado (nivel examen):**
- *muy orgulloso de su trabajo* → MOD (adv) + N + MOD (SPrep)
- *bastante harto de esperar* → MOD (adv) + N + MOD (SPrep)

**Análisis del MOD cuando es SPrep:**
- *de su trabajo* → SPrep: E (*de*) + T (*su trabajo*, que es un SN: Det + N)

---

## 3. Sintagma Adverbial (SAdv) ▲

### Definición

El sintagma adverbial tiene como núcleo un **adverbio**. Indica circunstancias (tiempo, lugar, modo, cantidad...).

### Estructura

| Componente | Abreviatura | Qué es | Ejemplos |
|------------|-------------|--------|----------|
| Núcleo | **N** | Adverbio (palabra principal) | lejos, ayer, bien, mucho |
| Modificador | **MOD** | Cualquier elemento que acompaña al adverbio | muy, bastante, de aquí |

### Fórmula

```
SAdv = (MOD) + N + (MOD)
```

### Tipos de modificadores en el SAdv

El modificador puede ser:
- **Otro adverbio:** *muy lejos* → MOD = adverbio
- **Un SPrep:** *cerca de mi casa* → MOD = SPrep

### Ejemplos graduados

**Básico:**
- *lejos* → N
- *ayer* → N

**Intermedio:**
- *muy lejos* → MOD (adv) + N
- *bastante bien* → MOD (adv) + N

**Avanzado (nivel examen):**
- *muy lejos de aquí* → MOD (adv) + N + MOD (SPrep)
- *cerca de mi casa* → N + MOD (SPrep)

**Análisis del MOD cuando es SPrep:**
- *de mi casa* → SPrep: E (*de*) + T (*mi casa*, que es un SN: Det + N)

---

## 4. Sintagma Preposicional (SPrep) ◆

### Definición

El sintagma preposicional comienza siempre por una **preposición** seguida de un **sintagma nominal** (o equivalente). La preposición es el **enlace** y el SN es el **término**.

### Estructura

| Componente | Abreviatura | Qué es | Ejemplos de preposiciones |
|------------|-------------|--------|---------------------------|
| Enlace | **E** | Preposición que introduce | a, ante, bajo, con, de, desde, en, entre, hacia, hasta, para, por, según, sin, sobre, tras |
| Término | **T** | SN que sigue a la preposición | la mesa, mi casa, él |

### Fórmula

```
SPrep = E (preposición) + T (SN)
```

**Importante:** El término siempre es un SN, así que se analiza como tal (Det + N + MOD...).

### Ejemplos graduados

**Básico:**
- *en casa* → E + T (N)
- *de madera* → E + T (N)

**Intermedio:**
- *para mi hermano* → E + T (Det + N)
- *con mucho cuidado* → E + T (Det + MOD + N)

**Avanzado (nivel examen):**
- *desde el parque grande* → E + T (Det + N + MOD)
- *por la mañana temprana* → E + T (Det + N + MOD)

## Consejos para el examen

### 1. Localizar sintagmas

1. **Identifica el núcleo primero**: ¿Es sustantivo, adjetivo, adverbio o preposición?
2. **Busca los límites**: ¿Dónde empieza y termina el sintagma?
3. **Comprueba**: Todas las palabras deben depender del núcleo.

### 2. Analizar la estructura interna

1. **Subraya el núcleo** (N).
2. **Identifica el determinante** (Det) si hay sustantivo.
3. **Busca modificadores** (MOD) y di de qué tipo son (adj, SPrep, SAdv...).
4. **Si hay SPrep**, analiza su estructura interna (E + T).

### 3. Errores frecuentes

| Error | Corrección |
|-------|------------|
| Confundir el tipo de sintagma | Fíjate en el núcleo, no en los modificadores |
| Olvidar analizar el MOD | Si el MOD es un SPrep, analiza también E + T |
| No ver el determinante | *el*, *la*, *un*, *mi*, *este*... siempre van con sustantivo |

## Versión redactada (para quien prefiere leer)

### El sintagma nominal

El sintagma nominal es el más importante y frecuente. Su núcleo es siempre un sustantivo (o un pronombre que lo sustituye). Delante del sustantivo puede aparecer un determinante (*el*, *mi*, *tres*...) y alrededor del núcleo pueden añadirse modificadores de distintos tipos.

Por ejemplo, en *el coche rojo de mi padre*, el núcleo es *coche* (sustantivo), *el* es el determinante, *rojo* es un modificador (un adjetivo) y *de mi padre* es otro modificador (un sintagma preposicional). Si analizamos ese SPrep, vemos que *de* es el enlace y *mi padre* es el término (un SN formado por Det + N).

### El sintagma adjetival

El sintagma adjetival tiene como núcleo un adjetivo. Puede ir solo (*alto*, *feliz*) o acompañado de modificadores. Si el modificador es un adverbio, tendremos *muy alto* (MOD + N). Si el modificador es un sintagma preposicional, tendremos *orgulloso de su trabajo* (N + MOD).

### El sintagma adverbial

El sintagma adverbial tiene como núcleo un adverbio (palabras como *lejos*, *ayer*, *bien*, *mucho*...). También puede llevar modificadores: otro adverbio (*muy lejos*) o un sintagma preposicional (*cerca de aquí*).

### El sintagma preposicional

El sintagma preposicional es especial: siempre empieza por una preposición (*a*, *de*, *en*, *con*, *para*, *por*...) seguida de un sintagma nominal. La preposición es el enlace (E) y el sintagma nominal que sigue es el término (T). Por eso *en la mesa* es un SPrep donde *en* es el enlace y *la mesa* es el término.
